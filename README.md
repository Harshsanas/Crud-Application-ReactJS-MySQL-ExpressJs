# Crud-Application-ReactJS-MySQL-ExpressJs
Crud Application using react js, express js, mysql database


To Clone this repository 

git clone https://github.com/Harshsanas/Crud-Application-ReactJS-MySQL-ExpressJs.git

Cd Server

npm i to install node mudules

npm run devStart to start server

cd ..

cd clientView

npm i to install node mudules

npm start to start application
